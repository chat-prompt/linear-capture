## 워크플로우

```
01-계획 → 02-실행 → 03-완료 → 04-학습
  계획      실행     결과 보관    교훈 기록
```

- **새 작업**: `01-계획/`에서 선택 → `02-실행/`으로 이동
- **작업 완료**: `02-실행/`에서 `03-완료/`에 기록 추가
- **교훈 기록**: `04-학습/F##-*.md` 추가
- **기능 추적**: `find .sisyphus -name "F02-*"` → 검색 관련 전체 조회
